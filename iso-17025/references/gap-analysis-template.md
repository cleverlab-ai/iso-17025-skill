# ISO/IEC 17025:2017 — Gap Analysis Template

## How to Use

For each requirement, assess the current state and assign a compliance level:

| Level | Status | Description |
|-------|--------|-------------|
| C | Conforming | Requirement fully met with evidence |
| PC | Partially Conforming | Some elements in place but gaps exist |
| NC | Non-Conforming | Requirement not met or no evidence |
| NA | Not Applicable | Requirement does not apply to this laboratory |

## Priority Levels
- **HIGH** — Must be addressed before accreditation/audit (non-negotiable requirements)
- **MEDIUM** — Should be addressed soon (gap could lead to finding)
- **LOW** — Minor improvement needed (observation level)
